Stacks

Abstract data structures (LIFO)

Stacks:

  • Conceptual - Multiple ways of implementing a stack.
  • Last item inserted is the first removed.
  • Seen in managing function invocation (call stack).
  • Seen in undo/redo actions.
  • Routing (such as the history object in React) uses stack.
  • Arrays CAN function as stacks (push/pop or shift/unshift, but not mixed)

Big O

  • Insertion - O(1)
  • Removal - O(1)
  • Search - O(N)
  • Access - O(N)
